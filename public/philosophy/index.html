Philosophy

Mission / Vision / Core Values

Mission

개인과 기업 모두 비용 부담없이 안전한 ICT 시스템의 구축 및 운용

Vision

  • 턴키 서비스 체계 구축 - 2020년
  • 500개 개업과 개인에 대한 서비스 - 2030년

Core Values

  • ICT 전 분야에 걸친 종합적인 상담 / 시스템 구축과 운용
  • 장비(Hardware)가 아닌 지식과 기술을 판매
  • 고객 중심 서비스 - 고객이 구매상품을 이해하도록 컨설팅
